Install Valet

Takes about 10 seconds

1
Open Terminal — press Cmd+Space, type Terminal, press Enter
2
Click the box below to copy the install command, then paste it in Terminal with Cmd+V and press Enter
curl -fsSL https://raw.githubusercontent.com/bottarjames/Valet/main/install.sh | bash
3
In Chrome → chrome://extensions → enable Developer mode → Load unpacked → press Cmd+Shift+G → paste ~/.valet/extension

Everything runs locally on your machine. No accounts, no cloud, no tracking.
